Ch2: Programming basics: Variables, scope, converting/parsing, and template literals.

Const is used when the variable will not be reassigned to another value.

Let is used when the variable will be reassigned later in the program.

Let should be used instead of var because let has block scope. Var can be used in old code

I can use camelCased or underscore_separated variable names

In the following code, changing d.value also changes c.value because d and c reference the same object:


const c = { value: 1 };
let d = c; // c.value = 1, d.value = 1
d.value = 2; // c.value = 2, d.value = 2
  • String functions
    • .toUpperCase() : string
    • .toLowerCase() : string
    • .charAt(index) : char
    • .indexOf(character) : number
    • .lastIndexOf(character) : number
    • .startsWith(character) : bool

    • Arrays, Logic, and Loops: array methods.

    Non-primitive data types, such as arrays, functions and objects all have a type of "Object".

  • Arrays Properties
    • Length: is used as part of the index to find the last item in an array.
    • Pop: Is used to remove the last item in an array.
    • Push: appends a new value to the end of the array.
    • Shift: remove the first item in array.
    • UnShift: appends a new item to the beginning of the array.
  • Arrays Methods
    • concat() : can be used to merge an array with one or more arrays.
    • join() : can be used to turn the array into a string that comprises all items in the array(separated by commas).
    • slice(): creates a subarray.
    • splice(): removes items from an array then inserts new items.
    • reverse(): reverse the order of an array.
    • sort(): sort the order of an array.
    • indexOf(): find out if an array contains a particular value
    • includes(): returns a boolean value depending on whether the arrays contains a particular element or not.
