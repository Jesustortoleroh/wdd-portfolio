In order to highlight what is most important about local storage I will answer the following questions

1. How can I store data so it isnâ€™t lost when the application stops?

When applications stop, they do not store information and it is lost. Applications must be created to store information,

2. How do I get data back into my application once it has been stored?

The information stored or shared with the user is called output. Anything coming in from where it was stored or anything the user enters is called input. Using JavaScript I can store any information I want to use later.

3. What can I do to allow the user to communicate?

An application allows people to write, save, edit or show how to store written information.

Communication is vital for us, one of the most important verbs is to remember. To remember it is necessary to create applications to store the information.

function loadStory(){ var storyName = document.getElementById("name_input").value var storyHTML = localStorage.getItem("storyName") document.getElementById("story_editor").value = storyHTML } function saveStory(){ var storyName = document.getElementById("name_input").value var storyHTML = document.getElementById("story_editor").value localStorage.setItem(storyName, storyHTML) } function displayStory(){ var storyHTML = document.getElementById("story_editor").value document.getElementById("story_display").innerHTML = storyHTML }
