Forms

document.forms will return an HTML collection of all forms on a page.

form.element will return an HTML collection of all the elements contained within a form.

Forms have focus, blur, and change events that fire as a user interacts with the form.

HTML5 form validation API is used to validate a form but at a basic level.

Forms Control

input fields: text, passwords, check boxes, radio buttons and files uploads

select menus: drop-down lists of options

textarea: longer text entry

submit and reset forms

Data Validation

Server side validation is performed by a web server, after input has been sent to the server.

Client side validation is performed by a web browser, before input is sent to a web server.

Object Oriented Programming

Three main concepts in OOP: Encapsulation, polymorphism and inheritance.

Constructor functions is used to create instances of objects.

The keyword this refers to a object returned by the function.

Moderm Javascript

Modular Javascript: A Module provides functions and methods that can then be used by other files and modules.

Modules allow a public API to be exposed, while keeping the implementation hidden away inside the module
