The Window Object

The Window Object is the browser window that contains a web page.

The Browser Object Model [BOM] is a collection of properties and methods that contain information about browser and computer screen.

Global Variables are properties of the window object in a browser.

Window object methods:

  • The window.alert() method will pause the execution of the program and display a message in a dialog box.
  • The window.confirm() method will stop the execution of the program and display a confirmation dialog that shows the message provided as an argument, and giving the options of OK or Cancel.
  • The window.prompt() method will stop the execution of the program, and then shows a message and an imput field that allows the user to enter text.

Window object properties:

  • The window.navigator object gives information about the user´s browser and operating system. This can be unreliable.
  • The window.location object provides information about the URL of the current page.
      • Location Properties:
    • href returns the full URL as a string.
    • protocol returns a string describing the protocol used. (http, https, pop2, ftp).
    • host returns a string describing the domain of the current URL and the port number.
    • hostname returns a string describing the domain of the current URL.
    • port returns a string describing the port number.
    • pathname returns a string of the path that follows the domain.
    • search returns a string that starts with a "?" followed by the query string parameters.
    • hash returns a string that starts with a "#" followed by the fragment identifier.
    • origin returns a string that shows the protocol and domain where the current page originated from.
      Location Methods:
    • reload() can be used to force a reload of the current page.
    • assign() can be used to load another resource from a URL provided as a parameters.
    • replace() is the same as assign() method, but the current page will not be stored in the session history.
    • toString() returns a string containing the whole URL.
  • The window.history object keeps information about the pages that have been visited in the session.
  • The window.screen object contains information about the screen the browser.

HTML5 APIs.

data- attributes help to embed custom data into a web page that is used to enhace the user with JavaScript.

Examples:

data-powers = "flight superSpeed"

data-rating = "5"

The WEB STORAGE API allows key-value pairs to be stored on the user´s device in a similar ways to cookies, but without the same storage restrictions.

The Geolocation API allows you access the geographic coordinates of the user´s device. The user gives permission.

The WEB Worker API can be used to perform tasks in background. It helps to avoid unresponsive websites.
