Testing and Debugging

Errors are caused by System, Programmer or User.

Exceptions are an errors that produces a return value that can be used by the program.

Exceptions can be thrown using the thrown statement.

Stack Trace is a sequence of functions or methods called that is lead to the point where the error ocurred.

Warnings: A error that is not enough to cause a program to crash. The program will continue.

Linting Tools: JS LINT, JS HINT, ES LINT.

Error Objects are created by the host environment when an Exceptions occurs or using a constructor function.
