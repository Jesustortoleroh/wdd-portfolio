Further Functions

Functions have built-in properties as length and can have custom properties added.

call() and apply() methods can invoked a funtion with the value of this bound to an object as an argument.

Callback is a function, this is provided as an argument to another function and is used in asynchronic programming as part of event loop.

Promises are used instead of Callbacks to deal with multiple asynchronic actions in sequence.

AJAX

AJAX is a technique that allows web pages to communicate asynchronously with a server, and dynamically updates web pages without reloading.

API:Application programming interface is a collection of methods that allows external access to another program or service.

Fetch API provides a global fetch() methods with a mandatory argument, this is the URL of the wished resource to fetch.
